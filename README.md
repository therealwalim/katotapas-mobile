# Mobile Application

## Summary

> This application aims to give access to the Katotapas API data on a mobile interface, to access it you will have to authenticate yourself as the transactions require a token.

### Tools used :
* **Kotlin**
* **Retrofit**
* **Lottie Files**
* **Picasso**
* **Coroutines**

## Functionalities

* **Splash Screen**
* **Login**
* **Session Management**
* **Call To API**
* **Product Display**
